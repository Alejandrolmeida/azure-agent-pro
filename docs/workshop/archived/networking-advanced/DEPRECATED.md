# ⚠️ Workshop Deprecado - Azure Networking con GitHub Copilot

**Estado:** DEPRECATED 
**Fecha de deprecación:** Diciembre 2025 
**Razón:** Reemplazado por contenido de mayor calidad y enfoque pedagógico mejorado

---

## 📢 Aviso Importante

Este workshop ha sido **archivado** y ya no se mantiene activamente. 

### ¿Por qué fue deprecado?

1. **Enfoque desactualizado**: Este workshop se centraba en usar MCP servers directamente, sin aprovechar el agente personalizado **Azure_Architect_Pro** que representa la propuesta de valor principal del proyecto.

2. **Barrera de entrada alta**: Requería conocimientos avanzados (AZ-104, AZ-700), limitando la audiencia.

3. **Pedagogía mejorada**: El nuevo workshop utiliza un enfoque progresivo y un tema divertido que facilita el aprendizaje.

4. **Contenido redundante**: Muchos conceptos están mejor explicados en el nuevo workshop.

---

## ¿Qué deberías usar en su lugar?

### Workshop Recomendado: **Kitten Space Missions API**

👉 **[Ver Workshop: Kitten Space Missions](../kitten-space-missions/)**

**¿Por qué es mejor?**

- **Nivel básico**: Accesible para principiantes
- **Vibe Coding**: Aprende a trabajar con el agente Azure_Architect_Pro

- **Completo**: FinOps, Bicep, CI/CD, Security, Monitoring, Testing
- **Divertido**: Tema de gatitos astronautas que hace el aprendizaje memorable
- **Production-ready**: Código y prácticas listas para usar en proyectos reales

**Cubre los mismos conceptos de networking** (VNets, NSGs, Private Endpoints) pero de forma más didáctica y moderna.
