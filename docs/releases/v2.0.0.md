# Release v2.0.0 - Azure Agent Pro v2

**Release Date:** July 3, 2026 
**Type:** Major Release

---

## Highlights

Azure Agent Pro v2 convierte el proyecto en un ecosistema mucho más fuerte para **VS Code + WSL**, con 7 agentes especializados, mejor hardening y un setup reproducible sin secretos ni datos personales en el repo.

### Nuevas capacidades

1. **5 sub-agentes especializados nuevos**
 - Azure_Admin_Pro
 - Azure_Data_Pro
 - Azure_AppServices_Pro
 - Azure_Foundry_Pro
 - Azure_Networking_Pro

2. **Azure MCP corregido y endurecido**
 - `@azure/mcp@latest` como package correcto
 - `MEMORY_FILE_PATH` para memoria persistente
 - Configuración pública sin credenciales hardcoded

3. **Setup WSL-first**
 - Script interactivo `scripts/setup/setup-wsl.sh`
 - Variables `.env.example` listas para cada equipo
 - Flujo pensado para VS Code sobre WSL/Linux

4. **Documentación de referencia nueva**
 - `docs/reference/agents-overview.md`
 - `docs/getting-started/mcp-quickstart.md`
 - `.copilot/copilot-instructions.md`

5. **Memoria persistente**
 - `memory-mcp` integrado con `MEMORY_FILE_PATH`
 - Contexto entre sesiones sin comprometer datos sensibles

6. **Onboarding mejorado**
 - Setup interactivo para WSL/Linux
 - `.env.example` completo y portable
 - Flujo pensado para clonar, configurar y usar rápido

---

## Mejoras incluidas en esta versión

### MCP y tooling

- `mcp.json` corregido para usar `@azure/mcp@latest`
- Eliminación de la configuración antigua/incompatible del servidor Azure MCP
- `memory-mcp` preparado para persistencia local
- Descripciones más claras para cada MCP server
- Flujo public-safe sin credenciales hardcoded

### Sub-agentes especializados

- `azure-admin.agent.md`
 - Governance, Policy, RBAC/PIM, Entra ID, Defender, Cost Management
- `azure-data.agent.md`
 - SQL, Cosmos DB, Synapse, Data Factory, Databricks, ADLS, Purview
- `azure-app-services.agent.md`
 - App Service, Functions, Container Apps, AKS, APIM, Service Bus
- `azure-foundry.agent.md`
 - Azure OpenAI, AI Foundry, Prompt Flow, AI Search, RAG, Responsible AI
- `azure-networking.agent.md`
 - VNets, NSGs, Azure Firewall, VPN/ER, Private Link, hub-spoke
- `azure-architect.agent.md`
 - Frontmatter modernizado y tabla de delegación a sub-agentes

### Documentación

- `.copilot/copilot-instructions.md` con metodología enterprise completa
- `docs/reference/agents-overview.md` como guía maestra de los 7 agentes
- `docs/getting-started/mcp-quickstart.md` reescrito para WSL-first
- `README.md` actualizado con la nueva propuesta de valor v2
- `CHANGELOG.md` y `RELEASE_NOTES.md` alineados con la nueva versión

### Setup y hardening

- `scripts/setup/setup-wsl.sh` para setup guiado en WSL/Linux
- `.env.example` ampliado con variables para Azure OpenAI, AI Search y memoria
- Recomendación explícita de VS Code + WSL como entorno preferido
- Sin datos personales, secretos ni configuraciones privadas en el repo público

## Security & Hardening

- No passwords, tokens or personal configs committed
- Public repo safe by default
- Setup local por equipo
- WSL-first guidance for reproducible onboarding
- Azure auth y GitHub auth resueltas en cada máquina por el usuario
- Sin dependencia de valores sensibles versionados
